# Plan 004: Points-to Analysis (E4)

## Overview

Implement Andersen-style context-insensitive points-to analysis with configurable field sensitivity. Operates on AIR, produces deterministic results, exposes query API for downstream analyses.

## Requirements Coverage

- FR-PTA-001: Andersen-style inclusion-based, context-insensitive PTA with bounded field sensitivity
- FR-PTA-002: Constraint extraction (Addr/Copy/Load/Store/GEP)
- FR-PTA-003: Deterministic solver (stable worklist, stable constraint processing)
- FR-PTA-004: Query API (`points_to`, `may_alias`)

## Design Decisions

| Topic | Decision | Future Extension |
|-------|----------|------------------|

## Constraint Extraction Rules

| AIR Operation | Constraint | Notes |
|---------------|------------|-------|
| `Alloca` | `Addr(dst, loc)` | Create new location for alloca |
| `Global { obj }` | `Addr(dst, loc)` | Location for global |
| `HeapAlloc` | `Addr(dst, loc)` | Location for heap site |
| `Load` | `Load(dst, ptr)` | ptr is operand[0] |
| `Store` | `Store(ptr, val)` | ptr is operand[1], val is operand[0] |
| `Gep { field_path }` | `Gep(dst, ptr, path)` | ptr is operand[0] |
| `Cast` | `Copy(dst, src)` | Pointer casts propagate |
| `Copy` | `Copy(dst, src)` | Direct copy |
| `Phi { incoming }` | `Copy(dst, val)` for each | Merge from predecessors |
| `Select` | `Copy(dst, true_val)`, `Copy(dst, false_val)` | Both branches |
| `CallDirect { callee }` | `Copy(param_i, arg_i)` | For each argument |
| `CallDirect` (return) | `Copy(dst, ret_val)` | If function returns pointer |
| `CallIndirect` | Conservative/placeholder | Deferred to future |

## Solver Algorithm

```
function solve(constraints, locations) -> pts:
    pts = BTreeMap::new()
    worklist = BTreeSet::new()

    // Initialize from Addr constraints
    for Addr(p, loc) in constraints.addr:
        pts[p].insert(loc)
        worklist.insert(p)

    // Fixed-point iteration
    iterations = 0
    while !worklist.is_empty() && iterations < max_iterations:
        iterations += 1
        v = worklist.pop_first()  // Deterministic: smallest ValueId

        // Process Copy constraints where v is source
        for Copy(dst, src) in constraints.copy where src == v:
            if pts[dst].union_changed(pts[v]):
                worklist.insert(dst)

        // Process Load constraints where v is pointer
        for Load(dst, ptr) in constraints.load where ptr == v:
            for loc in pts[v]:
                if pts[dst].union_changed(pts[loc]):
                    worklist.insert(dst)

        // Process Store constraints where v is pointer
        for Store(ptr, src) in constraints.store where ptr == v:
            for loc in pts[v]:
                if pts[loc].union_changed(pts[src]):
                    worklist.insert(loc)

        // Process Gep constraints where v is source pointer
        for Gep(dst, ptr, path) in constraints.gep where ptr == v:
            for loc in pts[v]:
                new_loc = locations.get_or_create(loc.obj, loc.path.extend(path))
                if pts[dst].insert(new_loc):
                    worklist.insert(dst)

    return pts
```
