# Getting Started with SAF

This category provides your first contact with SAF's core capabilities. All tutorials are beginner-level and introduce fundamental concepts.

## Learning Objectives

After completing these tutorials, you will be able to:
- Detect basic taint flows from sources to sinks
- Visualize program structure (call graphs, CFGs)
- Understand data flow concepts (def-use chains, value flow)
- Write a simple memory leak checker

## Tutorials

| # | Tutorial | Difficulty | Focus |
|---|----------|------------|-------|
| 01 | [Hello Taint](01-hello-taint/) | Beginner | Your first taint flow detection |
| 02 | [Call Graph & CFG](02-call-graph-cfg/) | Beginner | Visualize program structure |
| 03 | [DefUse & ValueFlow](03-defuse-valueflow/) | Beginner | Introduction to data flow |
| 04 | [Your First Checker](04-your-first-checker/) | Beginner | Simple memory leak detection |

## Recommended Order

Work through tutorials 01-04 in order. Each builds on concepts from the previous one.
